Easy User Tags adds a WordPress custom taxonomy called User Tags.

== Description ==

Easy User Tags lets you group your users/customers by a custom taxonomy called **User Tags**.

- Bulk add / remove tags from users
- Filter by tag in the user list view
- CSV export a list of users by tag


Easy User Tags integrates with AutomateWoo <http://automatewoo.com>.


== Screenshots ==

1. **User List View** - Filter users by tag and bulk add/remove tags from users.
2. **Taxonomy View** - Familiar WordPress Custom Taxonomy UI.


== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Easy User Tags'
3. Activate Easy User Tags from your Plugins page.

= From WordPress.org =

1. Download Easy User Tags.
2. Upload the 'easy-user-tags' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Easy User Tags from your Plugins page.
